Accessing External APIs

This section shows off three AngularJS features:

  • The $http service

    This asynchronously retrieves JSON data from an external API (i.e. a web service called Dbpedia).

  • Expressions

    These are used in this example to enhance the user interface by toggling the display of various HTML elements depending on $scope factors.

  • Two-way binding to a model

Submitting the form will result in a GET request to the Dbpedia API, which returns archived Wikipedia articles connected to the keyword: "{{query}}".

Two-way binding allows ng-model to update in real-time based on the value of the input box.
