Directives

Roughly speaking, directives are used to create custom HTML web components (vis. creating domain-specific languages for web applications), and are expressed on the View through custom elements, attributes, and classes.

Clicking the icon to the right will allow a custom directive called
toggleIcon
to interact with the controller scope, in such a way that the header's color will change.
The
toggleIcon
directive looks like this: 
app.directive('toggleIcon', function () {
    return {
        restrict: 'EA',
        scope: false,
        link: function (scope, element, attrs) {
            element.bind('click', function () {
                // do things with controller scope, the element, and its attributes
            });
        },
        template: '<i ng-style="{\'color\': titleColor}" class="fa fa-toggle-off fa-3x"></i>',
        replace: true
    }
});
